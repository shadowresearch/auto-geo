# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating — opening issues, submitting PRs, joining discussions — you agree to abide by its terms.

## Reporting

To report a Code of Conduct concern, email **conduct@shadow.inc**. Reports are confidential and reviewed by the maintainers. We aim to respond within 5 business days.

## Enforcement

The maintainers will follow the [Community Impact Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) in determining consequences for any action they deem in violation of this Code of Conduct.
